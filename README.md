## 🧠 Hakkımda | About Me

<img align="right" height="150" src="https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExNjJ5bHdybnZtd296Z2t3YXY5aW1pZnF2MTVpOG1qbGkzeHFzenZzcCZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/M9gbBd9nbDrOTu1Mqx/giphy.gif"/>

Full-stack web geliştirme ve UI/UX tasarımına olan tutkum, dijital dünyada etkileyici çözümler üretmemi sağlıyor. HTML, CSS, JavaScript ve TypeScript gibi front-end teknolojileri ile PHP, ASP.NET ve C# gibi back-end teknolojilerini kullanarak kapsamlı web çözümleri sunuyorum.

Kullanıcı deneyimini merkeze alan yaklaşımım, estetik olduğu kadar işlevsel de olan web uygulamaları geliştirmeme yardımcı oluyor. Şu anda Java ve Python öğrenerek teknoloji yelpazemi genişletiyorum.

My passion for full-stack web development and UI/UX design enables me to create impressive solutions in the digital world. I provide comprehensive web solutions using front-end technologies like HTML, CSS, JavaScript, and TypeScript, along with back-end technologies such as PHP, ASP.NET, and C#.

My user-centered approach helps me develop web applications that are as functional as they are aesthetic. I complete my projects to professional standards with database management, server optimization, and modern web development tools (Git, Docker). I'm currently expanding my technology range by learning Java and Python.
